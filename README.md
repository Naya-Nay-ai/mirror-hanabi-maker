# Mirror Hanabi Maker

大切な相手と一緒に見るための、共有できる花火メーカーです。

## v1 の内容

- Canvas の花火アニメーション
- 名前・相手の名前・呼びかけ名の入力
- 色 2 色の選択
- 花火タイプの選択
- 夜の雰囲気の選択
- 夜空に出したい言葉の入力
- 同じ設定を開ける共有 URL
- 動画を見られない相手にも渡せる「一緒に見るための文」
- スクショ向け表示
